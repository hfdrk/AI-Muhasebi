# Features Documentation

This directory contains detailed documentation for major features of the AI Muhasebi platform.

## Available Features

### [Global Search](./global-search.md)
Unified search interface for finding clients, invoices, documents, risk alerts, and reports across the system.

**Key Features**:
- Keyboard shortcut (Ctrl+K / Cmd+K)
- Multi-entity search
- RBAC-protected results
- Tenant-isolated

### [Saved Filters](./saved-filters.md)
Save and reuse filter configurations for list pages with default filter support.

**Key Features**:
- Save current filter state
- Set default filters
- Per-user, per-target organization
- Auto-apply default filters
